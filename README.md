# Project Management System (PMS)

A **Django-based Project Management System** to manage projects, tasks, and users with **role-based access**.

---

## Features

- **Role-based Users**: Admin, Manager, Employee.
- **Project Management**: Managers and Admins can create, update, and view projects.
- **Task Management**: 
  - Assign tasks to employees.
  - Employees can update the **status** of their tasks.
  - Managers and Admins can create, edit, and delete tasks.
- **Dashboard & Profile**: Users have personal profiles and dashboards.
- **Responsive UI**: Built with Bootstrap 5 for a clean interface.

---

## Technology Stack

- **Backend**: Django 5.2
- **Frontend**: HTML, CSS, Bootstrap 5
- **Database**: SQLite (default, can be changed)
- **Authentication**: Role-based user authentication

---
